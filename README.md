## About

Personal portfolio showcasing my machine learning projects, GitHub work, DSA progress and skills. Built with React, Vite and Tailwind CSS.

---

## Tech Used

- **React 19** + **Vite 8**
- **Tailwind CSS 4**
- **Framer Motion** — animations
- **EmailJS** — contact form
- **Deployed on Vercel**

---

## Sections

- **Hero** — typing animation, resume download, stats
- **About** — bio, skill breakdown with progress bars
- **Projects** — ML project cards with GitHub + live app links
- **GitHub** — all repos listed with language indicators
- **Education** — degree, certifications, GitHub activity graph
- **LeetCode** — DSA roadmap progress tracker
- **Contact** — working contact form (delivers to Gmail)

---

## Run Locally

```bash
git clone https://github.com/Bhumik2005/Portfolio.git
cd Portfolio
npm install
npm run dev
```

Open http://localhost:5173
